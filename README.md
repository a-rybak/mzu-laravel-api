## Installation

How to set up a project

- git clone a project
- run **composer update**
- run **php artisan migrate --seed --seeder=PurchaseSeeder**
- run **php artisan serve** to start a local web-server
- visit http://localhost:8000/docs/api/ and see what requests are available
